# Gerenciador de Jogos com SQLite e Python 🎮🐍

Este projeto foi desenvolvido com o objetivo de praticar conceitos de banco de dados relacionais utilizando Python com SQLite. A ideia de usar jogos como exemplo tornou o processo de aprendizado mais interessante e divertido.

## 📚 Sobre o Projeto

O sistema permite cadastrar:

- Consoles (com nome e fabricante)
- Jogos (com nome, gênero, preço e console ao qual pertence)
- Avaliações (com nota e comentário para cada jogo)

Tudo isso é feito com Python e armazenado em um banco de dados SQLite. Foi uma experiência muito válida para reforçar o uso de comandos SQL e integração com linguagens de programação.

## 🧠 Finalidade

Este projeto tem fins **educacionais** e foi criado como parte dos meus estudos em banco de dados. A proposta é simples, mas ajuda bastante a entender na prática como funciona a criação de tabelas, relacionamentos e inserção de dados.
